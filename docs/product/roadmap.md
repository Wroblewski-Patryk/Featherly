# Roadmap

## Phase 1 (Stabilized MVP Core)
- Goal: stabilize the existing localized CMS baseline and close major route/runtime gaps.
- Exit criteria:
  - public dynamic routes for page/blog/project are finalized,
  - forms runtime submission is validated end-to-end,
  - docs/planning/context are synchronized with implementation.

## Phase 2 (Editor and Content Experience)
- Goal: improve editorial velocity and consistency.
- Exit criteria:
  - richer block builder ergonomics and shared module contracts,
  - stronger taxonomy/category coverage,
  - improved module-level validation paths.

## Phase 3 (Operational Maturity)
- Goal: harden release and monitoring for live operations.
- Exit criteria:
  - repeatable release candidate checklist with evidence,
  - stronger post-release monitoring baseline,
  - documented rollback and incident workflow.
