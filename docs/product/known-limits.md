# Known Limits

## Public Dynamic Routing
- Area: public runtime
- Limit: dynamic page/blog/project route exposure is not fully finalized.
- Impact: some content flows are implemented in controller logic but not fully reachable.
- Workaround: use currently exposed localized routes for available views.
- Planned fix milestone: Phase 1 roadmap completion.

## Categories/Taxonomy Alignment
- Area: admin/content model
- Limit: category/taxonomy references require consistency validation against current models/controllers.
- Impact: potential incomplete behavior for category-related flows.
- Workaround: rely on validated modules (pages/posts/projects/forms/templates).
- Planned fix milestone: Phase 1 to Phase 2 transition.

## Advanced Timeline Editor
- Area: UX/editor tooling
- Limit: deep timeline/scene authoring workflow is documented but not fully productized.
- Impact: animation capabilities are present but advanced orchestration is limited.
- Workaround: use current block-level animation configuration.
- Planned fix milestone: Phase 2.
