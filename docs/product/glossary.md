# Glossary

## Terms
- Block Builder: visual editor used to compose content from JSON block structures.
- Renderer: runtime layer that transforms block JSON into rendered UI components.
- Template: reusable layout/content wrapper (`header`, `footer`, `sidebar`, `page`).
- Locale Middleware: middleware that resolves locale from route/URL/session/default and applies URL defaults.
- Translation Integrity Test: test that verifies translation keys used in code are represented in translation source.
- Content Architecture: shared model/controller conventions across `Page`, `Post`, and `Project`.
- Revision: historical content snapshot associated through morph relations.
- Task Board: execution state document in `.codex/context/TASK_BOARD.md`.
