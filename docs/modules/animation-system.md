# Module: Animation System

## Goal
Enable progressive-enhancement animations for block-based content.

## Current Capabilities
- Block-level animation config with trigger/preset/params.
- Optional timeline sequencing model.
- Runtime GSAP integration path.

## Accessibility and Safety
- Respect reduced-motion preferences.
- Keep content usable without JavaScript animation runtime.
- Avoid animation-only information delivery.
